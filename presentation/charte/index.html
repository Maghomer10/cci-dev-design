Pictogrammes

Les pictogrammes doivent être de style filaire.
2 variantes sont possibles en fonction de l'usage.

Version simple

Idéal pour les plus petits éléments ou pour les interfaces digitales.

Version développée

Cette version de pictogramme permet d'agrémenter divers documents.
Elle utilise des pictogrammes filaires plus complexes, accompagnés de touches de couleurs.
Les nuances de couleurs correspondent à la thématique sur laquelle on l'utilise.
